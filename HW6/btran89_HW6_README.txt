To open Part1 MVC:
- Open the solution located here:
	Part1/MVC/MVC.sln
- Right click the "View" project inside the Solution Explorer and select "Set as StartUp Project"
- Run the project by pushing "F5" or selecting the green play button at the top of Visual Studio.
- The website should now be viewable

To open Part2 MVC:
- Open the solution located here:
	Part2/MVCApplication/MVCApplication.sln
- Run the project by pushing "F5" or selecting the green play button at the top of Visual Studio.
- The website should now be viewable
