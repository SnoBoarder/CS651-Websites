Please do the following to load the website:
- Load the server:
	- Open the solution in Visual Studio 2015 located at "BtranWebServer\WcfService1\WcfService1.sln"
	- Run the solution by pressing "F5"
- Load the website:
	- Open another instance of Visual Studio 2015 and open the website by selecting the folder "BtranWebSite"
	- Run by pressing "F5"
	- The website will interact with the server that's running in the other Visual Studio instance
