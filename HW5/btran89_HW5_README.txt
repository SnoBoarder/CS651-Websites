To open the counting words website:
- Open Visual Studio 2015
- Select "File/Open/Web Site..."
- Select the "BtranCountingWordsWebsite" directory
- Once the project is loaded, run the project by pushing "F5" or selecting the green play button at the top of Visual Studio.
- The website should display in your specified browser.
